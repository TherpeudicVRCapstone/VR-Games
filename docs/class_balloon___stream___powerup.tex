\doxysubsection{Detailed Description}
The \doxylink{class_balloon}{Balloon} Stream Powerup class contains the logic for the balloon stream powerup balloon. The balloon stream powerup behaves like a regular balloon, but it spawn a stream of 5 more balloons when popped. The dart will not be destroyed to allow the stream to be popped easily.
