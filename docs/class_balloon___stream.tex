\doxysubsection{Detailed Description}
The \doxylink{class_balloon}{Balloon} Stream class contains the logic for the balloon stream. The balloon stream is simply a series of 5 regular balloons that are spawned simultaneously. These balloons do not destroy the dart so the player can pop the balloons in quick succession.

The Extra\+Pop\+Effects method is where extra effects (like adding extra lives for the life balloon) go. This tag is applied to the last balloon in the balloon stream. The parent object should be destroyed after this balloon is popped.

Destroys the parent object of the stream of balloons 3 seconds after the last balloon in the stream is popped. A delay is required to prevent the rest of the stream from being immediately destroyed if the last balloon is popped before some/all of the prior balloons in the stream.
